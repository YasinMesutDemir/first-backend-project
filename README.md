# first-backend-project

Ik heb een Website gebouwd waar de user de mogelijkheid heeft om te browsen door de webshhop en naar verschillende onderdelen kan kijken die hij/zij nodig zou hebben voor een computer, denk hierbij aan de onderdelen dat een werkende computer nodig heeft.

De gebruiker kan ook zelf onderdelen toevoegen aan de categorieen van de website. Het is bedoeld voor mensen die geintereseerd zijn in computer onderdelen en de belangrijke informatie willen zien van onderdelen/de prijs

# Installatie-instructies

Het installeren gaat als volgd, De repository downloaden en de files in een folder zetten. vervolgens de import.sql kopieren en in SQL terminal het plakken. De foto's moeten in de foto folder blijven. De talen HTML en PHP zijn gebruikt voor deze website en database MySQL
